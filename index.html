Getting started

Up and running in 2 minutes

No install required for telemetry analysis. Optional Python bridge for live session data.

Download the file
One HTML file. 540KB. Save it anywhere on your PC — Desktop, Documents, wherever. No installer, no setup wizard.
Open in Chrome or Edge
Double-click to open. Works completely offline. All analysis runs in your browser — nothing is uploaded anywhere.
Record your session
In iRacing, press Alt+L before leaving the pits. After your session find the file at Documents\iRacing\telemetry\
Drop your .ibt file
Drag it onto the Telemetry tab. All laps load instantly. AI coaching, tyre analysis, track map, corner breakdown — all update automatically.
Go live (optional)
Download the Python bridge from the Live tab. Run it alongside iRacing for real-time fuel, tyre and coaching data during your session.

HOW TO GO LIVE
1
DOWNLOAD BOTH FILES
Save both to the same folder. Desktop is fine.
2
RUN THE BRIDGE
Double-click pitwall_bridge_setup.bat
▸ Windows warning? → click More info → Run anyway
▸ Admin prompt? → click Yes
▸ Black window opens → keep it open while racing
3
ARM TELEMETRY IN iRACING
Press Alt + L before leaving the pits
A recording icon appears top-left — this arms full data capture
4
TEST THE CONNECTION
With the black window open:
⚠ TROUBLESHOOTING — tap to expand
"Windows protected your PC"
→ More info → Run anyway. The file is safe.

Black window opens then closes
→ Python not installed. Download from python.org
→ During install tick "Add Python to PATH"

Test connection fails
→ Check black window is still open
→ Wait 5 seconds, try again
→ Restart the .bat if needed

pip install failed
→ Check internet connection
→ Disable antivirus temporarily
→ Run .bat as Administrator

Bridge runs but no iRacing data
→ iRacing must be running and you must be on track
→ Press Alt+L in-car to arm telemetry

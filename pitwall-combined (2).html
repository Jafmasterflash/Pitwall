🔴 HOW TO GO LIVE

Option A — Python Bridge (full live data):
1. Download pitwall_bridge_setup.bat from below
2. Put it in the same folder as pitwall_bridge.py
3. Double-click pitwall_bridge_setup.bat — it installs everything automatically
4. Keep the black window open, come back here and click CONNECT

Option B — Auto-load last IBT (no Python needed):
1. Click GRANT FOLDER ACCESS on the left
2. Navigate to Documents\iRacing elemetry and click Select Folder
3. After each session, PIT WALL loads your latest file automatically

In iRacing: Press Alt+L before leaving the pits to arm telemetry recording.
